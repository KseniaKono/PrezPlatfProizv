Ficbook

  • Аудитория: Россия
  • Расположен: Латвия
  • Начало работы: 2009
  • Статус: заблокирован
  • На 2024 год является самой большой русскоязычной площадкой для публикации
  • Язык программирования: PHP

Fanfics

  • Аудитория: Россия
  • Начало работы: 2014
  • Статус: разблокирован
  • Помимо основной функции также является агрегатором текстов с других сайтов
  • Язык программирования: PHP

Litnet

  • Аудитория: Россия
  • Начало работы: 2015
  • Статус: доступен
  • Платформа-самиздат
  • Язык программирования: PHP

Author.Today

  • Аудитория: Россия
  • Начало работы: 2015
  • Статус: доступен
  • Упор на публикацию оригинальных произведений
  • Язык программирования: PHP (не точно)

Wattpad

  • Аудитория: многонациональная
  • Страна: Канада
  • Начало работы: 2006
  • Статус: доступен
  • Является полноценной компанией с офисом и штатом более 100 человек
  • Язык программирования: Ruby, Java

AO3

  • Аудитория: многонациональная
  • Начало работы: 2008
  • Статус: заблокирован
  • Некоммерческий с открытым исходным кодом, определяет себя как архив, а не как онлайн сообщество
  • Язык программирования: Ruby
